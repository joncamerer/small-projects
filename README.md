# small-projects

SudokuBoard - Generates a fully filled sudoku solution for 9 x 9 puzzles.

Unscramble - Idea taken from technical interview question.  The problem was to return an integer with digits in numerically ascending order derived from a scrambled string of the digits english names. For example "nofornineeu" should be represented as 149. Unscramble must be initialized with a scrambled word and a list of target words (provided). Unscramble.chaseSolution() will return the initial scrambled string and its representation as an integer.
